# BiMap — Frontend

The Angular client for BiMap — a geographic data entry application with an interactive map, role-based access control, and a polished Material + Bootstrap UI.

## 🚀 Development Server

Install dependencies and start the local dev server:

```bash
npm install
ng serve
```

Open your browser at `http://localhost:4200/`. The app reloads automatically on file changes.

## 🏗️ Code Scaffolding

Generate a new component with Angular CLI:

```bash
ng generate component component-name
```

List all available schematics:

```bash
ng generate --help
```

## 📦 Building

Build for production:

```bash
ng build
```

Artifacts are output to the `dist/` directory, optimised for performance and speed.

## 🧪 Running Unit Tests

Run unit tests with [Vitest](https://vitest.dev/):

```bash
ng test
```

## 🔬 Running End-to-End Tests

```bash
ng e2e
```

Angular CLI does not ship with an e2e framework by default — choose one that suits your needs.
